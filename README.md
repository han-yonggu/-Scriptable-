# [Scriptable] Latest-Youtube-Live-Shortcut
> IOS에서 Scriptable 앱을 이용하여 유튜브 채널의
> 최신 라이브 페이지로 이동하는 script입니다.

예정인 영상을 구분하지 않고 실시간 탭에 있는 리스트의 첫 번째 영상의 주소를 가져옵니다.

### 사용법
코드의 url 변수에 찾고자하는 Youtube 채널의 실시간 탭의 주소를 입력후 실행합니다.
